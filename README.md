# 🎟️ Event Ticket Booking System

## 📌 Project Overview

The Event Ticket Booking System is a modern and responsive web application that enables users to discover events, book tickets online, and manage their bookings with ease. It provides a seamless experience for both users and administrators through an intuitive interface and secure booking process.

---

## ✨ Features

### 👤 User Features
- User Registration & Login
- Secure Authentication
- Browse Available Events
- Search & Filter Events
- Event Categories
- View Event Details
- Real-Time Seat Selection
- Book Tickets Instantly
- Online Payment Integration
- QR Code Ticket Generation
- PDF Ticket Download
- Email Confirmation
- Booking History
- Cancel Bookings
- User Profile Management
- Responsive Design
- Dark Mode Support
- Notification System

### 🔧 Admin Features
- Secure Admin Login
- Admin Dashboard
- Add New Events
- Update Event Details
- Delete Events
- Manage Users
- Manage Bookings
- View Booking Statistics
- Event Analytics

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Event-Ticket-Booking.git
```

2. Open the project folder.

3. Configure Firebase (if required).

4. Launch the project in your browser or using Live Server.

---

## 💻 Usage

1. Create a new account or log in.
2. Browse available events.
3. Search or filter events.
4. Select your preferred event.
5. Choose seats.
6. Complete online payment.
7. Download your PDF ticket.
8. Scan the QR code at the event entrance.
9. Manage bookings from your profile.
